# Amar's YT Premium Auto-Skipper

A lightweight browser extension that automatically clicks YouTube's **"Jump Ahead"** prompt so you never have to.

>  **REQUIRES YouTube Premium** — The Jump Ahead feature is only available to YouTube Premium subscribers. This extension will NOT work without YT Premium.

---

## What it does

- Automatically clicks the "Jump Ahead" button when YouTube surfaces it
- Works without a problem across YouTube navigation without page reloads

## Installation

1. Download or clone this repo
2. Open extension page (`chrome://extensions`)
4. **Load unpacked** and select the project folder (must have developer mode on).
